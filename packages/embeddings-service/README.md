> 提供模型注入数据的搜索服务的插件

## 怎么使用？

1. 在插件市场安装本插件(`chathub-embeddings-service`)，并安装好本插件依赖的前置插件
2. 启用本插件即可
3. 就可以调用`chathub.listembeddings`，列举embeddings模型列表，再调用`chathub.setembeddings`，设置embeddings模型了。
