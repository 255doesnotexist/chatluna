> 提供模型注入数据的搜索服务的插件

## 怎么使用？

1. 在插件市场安装本插件，并安装好本插件依赖的前置插件
2. 启用本插件即可。
3. 尝试询问Bot一些有时效性的问题，对比开启和关闭插件后的回答，你会发现Bot的回答更加有时效性。
